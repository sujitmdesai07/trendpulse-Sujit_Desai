This project collects and analyzes trending stories from Hacker News to understand what topics are popular at the moment.

First, it fetches the top stories using an API and organizes them into categories like technology, science, sports, and entertainment. Then, the data is cleaned by removing duplicates, handling missing values, and filtering out low-quality posts.

Next, the project analyzes the data using Pandas and NumPy to find patterns such as average scores, most popular categories, and highly discussed stories. It also creates new metrics like engagement and popularity.

Finally, the results are visualized using charts (bar charts and scatter plots) and combined into a simple dashboard to make insights easy to understand.

This project demonstrates skills in API handling, data cleaning, data analysis, and data visualization using Python.
